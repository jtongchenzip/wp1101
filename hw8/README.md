## How to run the app
1. copy `.env.defaults`, rename it to `.env` and fill in the variables
2. `yarn install`  
3. `cd frontend`   
    `yarn install`   
    `yarn start`
4. open a differnet terminal    
    `cd backend`  
    `yarn install`  
    `cd ..`  (go back to `hw8`)  
    `yarn server`

Open [http://localhost:3000](http://localhost:3000) to view the app in the browser.
